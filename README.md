# ROMA TECH Website

Sitio web moderno y elegante con diseño dorado y animaciones avanzadas.

## 🚀 Instalación y Ejecución

### Requisitos Previos
- Node.js (versión 14 o superior)
- npm (viene con Node.js)

### Pasos para Ejecutar

1. **Descomprime el archivo ZIP** en tu carpeta deseada

2. **Abre una terminal** en la carpeta del proyecto

3. **Instala las dependencias:**
   ```bash
   npm install
   ```

4. **Inicia el servidor de desarrollo:**
   ```bash
   npm start
   ```

5. **Abre tu navegador** en [http://localhost:3000](http://localhost:3000)

¡Listo! Tu sitio web estará corriendo y podrás ver todos los efectos y animaciones.

## 📦 Compilar para Producción

Para crear una versión optimizada para producción:

```bash
npm run build
```

Los archivos compilados estarán en la carpeta `build/`.

## 💡 Personalización

Para modificar colores, edita las variables CSS en `src/App.css`:

```css
:root {
  --gold-primary: #D4AF37;
  --gold-light: #F4E6B8;
  --gold-dark: #B8941F;
  /* ... */
}
```
